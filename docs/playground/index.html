About Collie

Collie is a linter and formatter for Lrama Style BNF grammar files. It provides 18+ built-in lint rules to catch errors, enforce conventions, and suggest optimizations.

This playground runs completely in your browser using Ruby.wasm. Your code never leaves your browser.
